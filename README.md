JA全農杯 全国小学生選抜サッカー大会（ちびりんピック）2026 特設サイト プロトタイプ
本プロジェクトは、**『超ワールドサッカー』**プラットフォーム内での展開を想定した、JA全農杯 全国小学生選抜サッカー大会（ちびりんピック）の特設LPプロトタイプです。

📋 プロジェクト概要
小学生年代の最高峰の戦いである「ちびりんピック」の熱量を伝えるため、**「動画メインのUI」と「リアルタイムな情報集約」**を軸に構築しています。

ターゲット: ジュニアサッカーファン、出場選手保護者、指導者

デザインコンセプト: 超ワールドサッカーのダークモードを基調に、JA全農杯のフレッシュなグリーンをアクセントに使用

技術スタック: HTML5, CSS3 (Vanilla JSによる拡張を想定)

✨ 主要機能
1. ビデオ・ヒーローセクション
ページ最上部に公式YouTube配信やハイライト動画を配置。

大会期間中のライブ配信にも対応可能なアスペクト比固定レイアウト。

2. タグベースのコンテンツ集約
地域別（北海道〜九州）や大会フェーズ（予選・全国大会）ごとのタグナビゲーション。

ユーザーが見たい地域の情報に素早くアクセスできる導線。

3. インタラクティブ・スコアボード
全国大会のトーナメント結果や決勝スコアをカード形式で表示。

モバイル閲覧時も視認性の高いグリッドレイアウトを採用。

4. 出場クラブ SNSディレクトリ
全国大会出場チームの公式X、Instagramへのリンクをアイコン化。

各地域のジュニア強豪チームの「現場の熱量」へ直接アクセス可能。

🚀 プレビュー方法
本リポジトリをクローンまたはダウンロードします。

index.html を任意のブラウザで開きます。

（推奨）GitHub Pagesを有効にすることで、Web上で即座に確認可能です。
